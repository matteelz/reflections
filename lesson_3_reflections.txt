When would you want to use a remote repository rather than keeping all your work local?
Sharing/working with other people or for backup reasons.

Why might you want to always pull changes manually rather than having Git
automatically stay up-to-date with your remote repository?
Saving time and bandwidth, also focussing on only important commits avoid spam.

Describe the differences between forks, clones, and branches.  When would you
use one instead of another?
Forks to change an existing project for my own purposes and/or to work on with a
friend without bothering the initial author.
Clones to work on my local machine.
Branches for trying out new features that might or might not make it into the master.
